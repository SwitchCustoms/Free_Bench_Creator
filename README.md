Switch Crafting Bench
A modern, framework‑adaptive crafting system for FiveM with full support for QBCore, ESX, and Standalone. Powered by ox_inventory, ox_lib, and ox_target, it delivers a smooth, customizable crafting experience for any server.

Create, place, move, edit, and delete crafting benches entirely in‑game using an intuitive admin UI. Each bench supports unique recipes, job restrictions, animations, and crafting times — all synced through SQL for instant updates.

Perfect for weapon crafting, drug labs, mechanic workstations, black‑market crafting, and any custom gameplay loop.

🚀 Features

🔧 Unlimited Crafting Benches — Create, move, rename, and delete benches in‑game.

📦 Per‑Bench Recipes — Each bench has its own crafting list.

🛠 SQL‑Driven System — No restarts needed; everything updates live.

👮 Job‑Locked Crafting — Restrict benches to jobs or make them public.

🎨 NUI Crafting Interface — Clean, responsive UI via ox_lib.

🎬 Crafting Animations + Progress Circle — Supports custom anim dicts & clips.

🧰 ox_inventory Integration — Automatic item checks, removals, and rewards.

🎯 ox_target Interaction — Walk up, interact, and craft.

🔄 Framework Auto‑Detection — Works on ESX & QBCore without config edits.

🛡 Unified Admin System — Supports ESX groups, QBCore permissions, and ACE.
